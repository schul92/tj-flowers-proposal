01 — 현재 분석

현재 사이트 문제점

2016년에 구축된 OpenCart 기반 웹사이트의 핵심 문제를 진단했습니다.

항목 현재 (OpenCart) 마이그레이션 후 (Shopify)
플랫폼 위험 OpenCart 2016, 보안 업데이트 중단 → 안전 Shopify OS 2.0
모바일 최적화 ✘ 미지원 (고객 60%+ 이탈) → ✔ 완벽 반응형
보안 (SSL) ⚠ 취약한 SSL 관리 → ✔ Shopify 자동 SSL
결제 시스템 구식 결제 모듈 → Shopify Payments, Apple Pay, Google Pay
상품명 "TJflowers 21006" (코드명) → "Blush Splendor" (럭셔리 이름)
페이지 로드 ~5초 (느림) → ~1.5초 (3배 빠름)
SEO 낮음 구조화 데이터 없음 → 높음 Schema.org 완전 지원

04 — 새로운 기능

새롭게 추가되는 핵심 기능

현대적인 꽃집 웹사이트에 필요한 모든 기능을 구현했습니다.

🚚
당일 배달 공지바
주문 마감 시간을 상단에 실시간 표시. "오후 2시 전 주문 시 당일 배달" 안내로 전환율 향상.
📞
클릭 투 콜
모바일에서 전화번호 터치로 바로 연결. 헤더, 공지바, 연락처 페이지에 모두 적용.
💍
웨딩 & 이벤트 페이지
포트폴리오 갤러리, 6단계 프로세스 타임라인, 전용 문의 양식 (예산, 날짜, 장소, Pinterest 보드).
🖼️
갤러리 & 라이트박스
메이슨리 레이아웃 + 카테고리 필터링 + 키보드 네비게이션 라이트박스. 작품을 아름답게 전시.
📧
뉴스레터 구독
시즌별 영감, 독점 오퍼, 신규 컬렉션 소식 이메일 발송으로 고객 재방문 유도.
🗺️
Google 지도 통합
연락처 페이지에 인터랙티브 구글 지도 임베드. 그레이스케일 스타일로 럭셔리 분위기 유지.
⭐
리뷰 통합 (예정)
기존 3,937개 Google 리뷰를 웹사이트에 직접 표시. 경쟁사 Windsor(8,747개) 수준 도달 목표.
🤖
AI 상품 태그
Google Cloud Vision AI로 상품 이미지 자동 분석. 꽃 종류, 색상, 카테고리 자동 생성.
05 — 검색 최적화

SEO & 지역 검색 최적화

Google에서 "Upper East Side florist"를 검색할 때 TJ Flowers가 나타나도록 합니다.

✔
Schema.org LocalBusiness 구조화된 비즈니스 데이터 (주소, 전화번호, 영업시간, 좌표)를 Google에 직접 제공. 이미 구현 완료.
✔
키워드 타겟팅 "Upper East Side florist", "NYC flower delivery", "luxury flowers Manhattan" 등 고가치 키워드 최적화.
✔
자동 SEO 메타데이터 219개 모든 상품에 AI 생성 SEO 제목 + 설명 자동 적용. 수동 작업 제로.
✔
페이지 속도 3배 개선 ~5초 → ~1.5초. Google Core Web Vitals 기준 충족으로 검색 순위 상승.
✔
지리적 좌표 내장 40.7761732, -73.9488842 좌표가 코드에 내장. Google Maps 및 로컬 검색 노출 극대화.
✔
모바일 퍼스트 인덱싱 Google은 모바일 버전을 우선 색인. 완벽한 반응형 디자인으로 순위 불이익 제거.
06 — 마이그레이션 타임라인

실행 계획

4주 마이그레이션 플랜 + 지속적인 최적화.

Week 1
테마 개발 + 상품 데이터 정리
Shopify OS 2.0 커스텀 테마 제작, 219개 상품 스크래핑, AI 분석 및 이름 변환, CSV 준비.
✔ 완료
Week 2
상품 이미지 최적화 + 카테고리 설정
이미지 고화질 업로드, 컬렉션(Roses, Orchids, Plants 등) 구성, 메뉴 설정.
다음 단계
Week 3
결제/배송 설정 + 도메인 연결
Shopify Payments 설정, 배달 존 설정, tjflowersandevents.com 도메인 연결, SSL 자동 적용.
Week 4
테스트 + 런칭
전체 기능 QA, 모바일/데스크톱 테스트, 결제 테스트, 최종 런칭.
런칭 이후
지속적인 성장
Google 리뷰 수집, SEO 모니터링, 월간 퍼포먼스 리포트, 시즌별 컬렉션 업데이트.
